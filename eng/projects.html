Bachelor's Thesis

GitHub
Thesis

Spring 2019

The thesis suggests a learning method from narrative text by implementing two academic papers into software and combining their suggested narrative structure and learning models. This work involves concepts such as Narratology, Natural Language Processing, Unsupervised Learning, Algorithmic Optimisation and Procedural Generation. Applied Cryptography

GitHub

Spring 2019

This work consists of multiple implementations; AES-128 and RSA, Brute Force and Dictionary attack on passwords, Differential Power Analysis attack on the AES-128 implementation and taking countermeasures on the AES-128 implementation against the said DPA Attack. DPA Attack is made using PicoScope 2206B and SCALE kit. All implementations are programmed in C.
